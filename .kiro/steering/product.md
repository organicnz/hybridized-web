# Product Overview

Hybridized is a modern music discovery and exploration platform that combines unique hybrid sounds with interactive musical experiences.

## Core Features

- **Hybrid Sounds Library**: Browse and explore unique musical hybrids with formulas and embedded content
- **Interactive Tests**: Musical knowledge tests and sound discovery tools
- **User Authentication**: Supabase-powered auth with profile management
- **Contact & Donations**: Community engagement and support features

## Database Schema

### Tables

**hybridized**
- Stores hybrid music entries with formulas, descriptions, and media (iframe/cover URLs)

**profiles**
- User profiles linked to authentication with email, full name, and avatar

## Brand Identity

- Purple/pink/blue gradient color scheme
- Music-focused iconography (Music, Headphones, Sparkles)
- Modern, glassmorphic UI with backdrop blur effects
- Animated, interactive elements using Framer Motion
